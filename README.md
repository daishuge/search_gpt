# search_gpt2.0强势来袭!

## 简介
这是一个接入google 的chatgpt. 可以访问互联网

灵感来源于newbing

## 依赖

- Python 3.x
- BeautifulSoup
- googlesearch-python
- openai

## 安装

```bash
pip install beautifulsoup4 googlesearch-python openai
```

## 文件说明
gethtml.py: 用于获取网页HTML内容和鉴别输入中的url

geturl.py: google搜索得到排名第一url

main.py: 主程序文件,包含ai调用,输入处理
