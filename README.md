# search_gpt
## 简介
这是一个接入google 的chatgpt. 可以访问互联网
灵感来源于newbing

## 依赖

- Python 3.x
- BeautifulSoup
- googlesearch-python
- openai

## 安装

```bash
pip install beautifulsoup4 googlesearch-python openai
```

## 文件说明
cut.py: 用于文本切割

gethtml.py: 用于获取网页HTML内容

geturl.py: 用于获取搜索结果的URL

main.py: 主程序文件,包含ai调用
